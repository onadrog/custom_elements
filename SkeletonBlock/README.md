# SkeletonBlock

```html
<!-- html -->
<article class="card">
    <skeleton-block data-avatar="0" class="card-avatar" ></skeleton-block>
    <skeleton-block data-height="24" data-width="113px" ></skeleton-block>
    <skeleton-block data-lines="3" data-width="268" ></skeleton-block>
</article>
```
