# ThemeSwitcher

## Usage

```html
<theme-switcher>

    <label for="dark">Dark mode</label>
    <input
        type="radio"
        id="dark"
        name="theme-toggle"
        value="dark"
    />

    <label for="light">Light mode</label>
    <input
        type="radio"
        id="light"
        name="theme-toggle"
        value="light"
    />
</theme-switcher>
```
